Open, linked research on the Web

Ruben Verborgh

Ghent University – imec

The Semantic Web is doomed
if we don’t focus on the Web.

We need to practice
what we preach.

Our publication metadata needs to be
on the Web as Linked Data.

  • Self-publish your metadata.
    • We all know how to do it.
    • Others do it anyway, and they’re doing it wrong.
  • Publish semantic annotation with articles.
    • Link to other things.
  • Publishers are producers, harvesters, aggregators.
    • Not necessarily the (only) authorities.

I publish my own metadata
and you can query it.

  • All my publications are in my FOAF profile
    and annotated as RDFa in HTML.
  • They are published as Triple Pattern Fragments.
    • Browse my data.
    • Query my data.
  • Publish your data as well.
    • Then we can execute federated queries!
